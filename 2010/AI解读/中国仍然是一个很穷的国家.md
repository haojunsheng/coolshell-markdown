# 内容分析报告

## 一、核心内容
### 1. 核心论点
借助 Google Public Data 人均 GDP 可视化，在部分中国城市房价高企的舆论背景下提醒：中国人均 GDP 仍远低于美欧日等发达国家，中国「仍然是一个很穷的国家」。

### 2. 关键概念
| 概念 | 文中定义/内涵 |
|------|--------------|
| Google Public Data | 开放宏观数据集与交互图表平台 |
| 人均 GDP | 衡量国民平均财富水平的宏观指标，非收入分配 |
| 房价 vs 人均财富 | 作者对比「房价想象力」与「人均数据」的认知落差 |
| 多国对比曲线 | 图中含 CHN、USA、JPN、DEU 等 |

### 3. 文章结构
介绍 Google 数据平台 → 给出人均 GDP 探索链接 → 一句话核心观点 → 配图 we-are-still-poor.jpg。短评+数据可视化。

### 4. 案例与证据
- google.com/publicdata/directory
- 含中国等多国 idim 的 explore 链接
- 站内截图人均 GDP 对比图

## 二、背景语境
### 1. 作者身份
陈皓偶发社会/经济评论，与纯技术文形成反差，体现博客主个人视角。

### 2. 写作背景
2010 年 3 月，中国一线房价快速上涨引发焦虑；GDP 总量世界第二叙事流行。

### 3. 写作意图
校正「中国已经很富」的错觉；展示 Google 数据工具；引发读者自省。

### 4. 底层假设
人均 GDP 能代表「穷富」；读者应关心宏观指标；房价泡沫不推翻人均低位事实。

## 三、批判性审视
### 1. 主要反对意见
人均 GDP 掩盖基尼系数与城乡差距；购买力平价（PPP）未讨论；2010 年后中国人均大幅上升，结论需更新。

### 2. 论证漏洞
单指标简化；无收入分配、median 收入；房价讨论未深入成因。

### 3. 适用边界
适合宏观认知启蒙、辩论总量与人均；不适合政策分析或投资结论。

### 4. 回避问题
未讨论增长前景；未提区域差异；未分析数据可视化工具局限。

## 四、价值提取
### 1. 可复用思考框架
「总量光环」下必看人均与结构指标；数据新闻先用官方/开放数据验证直觉。

### 2. 对从业者的启发
技术人也应具备基本经济素养，避免将局部繁荣（互联网高薪）推广为全国现状。

### 3. 对决策者的启发
商业决策需分市场购买力分层；国际化产品勿假设统一消费力。

### 4. 认知升级点
2010 年的提醒在 2020 年代需用新数据重评，但「用人均校正自豪感」方法仍有效。
