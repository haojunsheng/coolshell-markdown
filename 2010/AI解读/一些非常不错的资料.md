## 二、背景语境
### 1. 作者身份
陈皓作质量把关型推荐人，坦言 jQuery 教程「没时间不然全翻译」。

### 2. 写作背景
2010 年 10 月，多核、jQuery 1.4、Windows 桌面与 Web 并存；免费在线书开始增多。

### 3. 写作意图
帮读者节省筛选成本；强化酷壳「资料宝库」定位；内链电子书系列。

### 4. 底层假设
英文资料对国内开发者仍高价值；Intel 书单代表工业界最佳实践；HtDP 适合「想学设计程序」读者。

## 三、批判性审视
### 1. 主要反对意见
jQuery 已衰落；HtDP 基于 Racket 生态，入门槛高；Intel 书单偏硬件/HPC 对 Web 开发者边际低；Codeplex 已关停。

### 2. 论证漏洞
「最好」无对比方法论；未说明各资料适合谁；无中文版指引。

### 3. 适用边界
适合资源考古、技术史；HtDP 仍适合 CS 教育；jQuery 仅适合遗留维护。

### 4. 回避问题
未讨论版权与离线保存；未提开源许可差异。

## 四、价值提取
### 1. 可复用思考框架
资料评价四维：权威性、示例密度、与当前栈匹配度、可维护性（是否仍更新）。

### 2. 对从业者的启发
大厂商书单可看趋势（2010 即重多核）；程序设计教材比语法教程更长期投资。

### 3. 对决策者的启发
团队图书馆应定期汰换；翻译/内训比个人「总有一天翻译」更可靠。

### 4. 认知升级点
2010 年「免费电子书+大厂清单」预示知识获取民主化，但策展能力更稀缺。
